Set up Cursor to work with HypurrFi documentation and integration code.

## Prerequisites

- Cursor editor installed

## Setup

<Steps>
<Step title="Open the docs directory in Cursor">
  Open the root of the documentation repository where `docs.json` is located.
</Step>

<Step title="Load the HypurrFi skill file">
  Add `SKILL.md` from the repo root to your Cursor context. This gives your agent the full HypurrFi protocol reference including product taxonomy, contract addresses, code patterns, and user interaction guidelines.
</Step>

<Step title="Review project instructions">
  `AGENTS.md` in the project root contains HypurrFi-specific terminology, style rules, and content structure. Cursor will read this automatically.
</Step>

<Step title="Start writing">
  Open a file and use Cursor's AI features to draft and edit documentation or integration code.
</Step>
</Steps>
