Set up Windsurf's Cascade AI assistant to work with HypurrFi documentation and integration code.

## Prerequisites

- Windsurf editor installed

## Setup

<Steps>
<Step title="Open the docs directory in Windsurf">
  Open the root of the documentation repository where `docs.json` is located.
</Step>

<Step title="Load the HypurrFi skill file">
  Add `SKILL.md` from the repo root to your Windsurf context. This gives Cascade the full HypurrFi protocol reference including product taxonomy, contract addresses, code patterns, and user interaction guidelines.
</Step>

<Step title="Review project instructions">
  `AGENTS.md` in the project root contains HypurrFi-specific terminology, style rules, and content structure. Cascade will read this automatically.
</Step>

<Step title="Start writing with Cascade">
  Open a file and use Cascade to draft and edit documentation or integration code.
</Step>
</Steps>
