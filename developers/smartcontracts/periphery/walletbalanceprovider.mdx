---
title: "WalletBalanceProvider"
---

Implements a logic of getting multiple tokens balance for one user address.

<Note>
  For getting HYPE (native chain token) balance use \`MOCK\_HYPE\_ADDRESS = 0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE\`.
</Note>

## Methods

### balanceOf

`function balanceOf(address user, address token)`

Returns the balance of the token for user (HYPE included with `MOCK_HYPE_ADDRESS`).

### batchBalanceOf

`function batchBalanceOf(address[] calldata users, address[] calldata tokens)`

Returns balances for a list of `users` and `tokens` (HYPE included with `MOCK_HYPE_ADDRESS`).

### getUserWalletBalances

`function getUserWalletBalances(address provider, address user)`

Provides balances of user wallet for all reserves available on the pool
